### Installation
$ git clone https://github.com/nail2019/nails
$ cd nails
$ npm install
$ npm run update-libs
$ npm start

nếu bạn muốn chỉ chạy  server  thì  dùng lệnh : **npm  run  start-server_**
ngược lại, nếu bạn chỉ muốn chạy client thì dùng lệnh: **npm run start-client_**
